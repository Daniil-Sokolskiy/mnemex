# Anna's Archive Disclaimer

This project includes an MCP client for Anna's Archive — a shadow library.

## Current legal context (as of 2026)

- Anna's Archive received a large default judgment in U.S. court, including a
  permanent injunction directing service providers to stop serving the site.
- As a result, Anna's mirror domains rotate frequently.

## What this project does NOT do

- It does **not** host or distribute any copyrighted books.
- It does **not** bypass any technical protection measure.
- It does **not** recommend any particular use case.

## What this project IS

- An HTTP/automation client for Anna's Archive's public search and download
  endpoints.
- A tool that **may** be used to download materials whose copyright status
  varies by jurisdiction — public domain in some countries, copyrighted in
  others.

## Your responsibilities

- You are responsible for complying with copyright law in your jurisdiction.
- Many books are in the public domain in some countries (everything on Project
  Gutenberg, for example). Using this tool for those is uncontroversial.
- For materials still under copyright, you should evaluate whether your use
  falls under fair use (US), fair dealing (UK/Commonwealth), or the equivalent
  in your country.
- The maintainers take no position on individual use cases and accept no
  liability for misuse.

## Copyright-clean alternative

If you prefer a fully copyright-clean workflow, use only the **Project
Gutenberg** tools bundled in `@mnemex/library-mcp`. They cover roughly 70,000
public-domain texts and require no third-party shadow library.
